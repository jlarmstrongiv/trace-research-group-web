# WIP: TRACE Research Group Website

## Installation

1. Clone the repository:

```sh
git clone git@github.com:jlarmstrongiv/trace-research-group.git
```

1. cd into the directory:

```sh
cd trace-research-group
```

1. Install the dependencies

```sh
npm install
```

1. Get started

```sh
npm run dev
```
